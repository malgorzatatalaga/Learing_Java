Halving Carousel

The purpose of this exercise is to train you to use extend classes with extends keyword.

Estimated workload of this exercise is 30 min.

Description
In this exercise you need to extend DecrementingCarousel. You need to implement HalvingCarousel.
This subclass must halve elements instead of decrementing it by one. Note that you need to apply regular integer division,
discarding the remainder. For example, 5 / 2 = 2.
